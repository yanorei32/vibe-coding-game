# ミニゲーム - 敵を避けてゴールへ

敵を避けながらゴールを目指すアクションゲームです。レベルが上がるごとに敵の数が増え、難易度が上がります。

## ゲームの概要

- プレイヤーを操作して、敵を避けながらゴールを目指します
- ゴールに到達すると次のレベルに進みます
- 敵に触れるとゲームオーバーです
- ハイスコア機能で到達した最高レベルを記録します

## 操作方法

- **矢印キー（↑↓←→）**: プレイヤーを移動
- **リセットボタン**: ゲームを最初からやり直す

## ゲームの特徴

### 敵の種類
- **静止した敵**: 動かない敵
- **動く敵**: ランダムな方向に移動する敵（30%の確率で生成）
  - 画面端で跳ね返ります
  - 安全地帯には侵入しません

### 安全地帯
- スタート地点周辺（左上）に安全地帯があります
- 敵は安全地帯に入ることができません

### レベルシステム
- レベル1からスタート
- ゴールに到達するたびにレベルが上がります
- レベルが上がるごとに敵の数が増えます（レベル1=1個、レベル2=2個...）

### ハイスコア
- 到達した最高レベルがブラウザのlocalStorageに保存されます
- リセットボタンでレベルを1に戻すことができます

3. ゲームを開始します

## ゲーム設定

ゲームの各種パラメータは`app.js`で設定できます：

- `GAME_WIDTH`: ゲームエリアの幅（デフォルト: 600px）
- `GAME_HEIGHT`: ゲームエリアの高さ（デフォルト: 400px）
- `PLAYER_SIZE`: プレイヤーのサイズ（デフォルト: 30px）
- `ENEMY_SIZE`: 敵のサイズ（デフォルト: 40px）
- `GOAL_SIZE`: ゴールのサイズ（デフォルト: 35px）
- `MIN_START_GOAL_DISTANCE`: スタートとゴールの最小距離（デフォルト: 350px）
- `MOVING_ENEMY_PROBABILITY`: 動く敵の生成確率（デフォルト: 0.3 = 30%）
- `MOVING_ENEMY_SPEED`: 動く敵の速度（デフォルト: 2）
- `moveSpeed`: プレイヤーの移動速度（デフォルト: 5）
